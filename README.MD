# Школограм

Обертка над преис... ой, Сетевым Городом для Telegram, планируется поддержка VK.

## Стэк технологий
Клиентская часть (код)
- React
- TypeScript
UI/UX (Telegram)
  - Telegram UI
  - Telegram Mini Apps SDK
UI/UX (VK)
  - VKUI
  - VK Bridge
  - VK Router
- Где-то Vite

### Структура проекта

- web - Frontend часть
- admin - Админская часть
- api - Backend часть (ПОО для Томской области)
- shared - общие типы
- api-*, *api - тщетные попытки получить API Томской области для работы с нормальным СГО 
- logintest - часть логина (тестовая, стырил у платного бота, не работает)
